# 行业知识 — 五金工具（Tools & Hardware）

## 核心品类方向
### 主力品类
- **电动工具（Power Tools）**：锂电钻、冲击起子、角磨机、电锤，锂电化是绝对主流。
- **手动工具（Hand Tools）**：扳手套装、螺丝刀、钳子、卷尺，属于刚需高频补货类。
- **紧固件（Fasteners）**：螺丝、螺母、膨胀螺栓，B端大宗采购的核心。
- **园林工具（Garden Tools）**：割草机、修枝剪、吹风机，季节性极强，锂电化趋势明显。
- **磨料磨具（Abrasives）**：切割片、砂轮、砂纸，工业消耗品，复购率极高。
- **测量工具（Measuring Tools）**：水平仪、激光测距仪、卷尺，精度和耐用性是卖点。

### 新兴/蓝海方向
- **无刷锂电系列（Brushless Cordless Tools）**：更高效率、更长寿命，正全面替代有刷电机。
- **智能激光测量（Smart Laser Measuring）**：带蓝牙连接App、自动绘图功能的测距仪。
- **多功能家装套装（DIY Home Toolkit）**：针对欧美居家维修潮的精致化、便携化组合包。

| 目标市场 | 必要认证 | 说明 |
|----------|---------|------|
| 欧盟 | CE / GS | GS是德国安全认证，在欧洲认可度极高，溢价能力强 |

## 行业避坑指南
1. **电池合规性**：锂电池属于第9类危险品，海运必须有MSDS、UN38.3报告，空运极其昂贵。
2. **插头适配**：电动工具务必确认目标市场的插头标准（美标/欧标/英标/澳标）及电压（110V/220V）。
3. **侵权风险**：外观专利极其严重，尤其是模仿大牌（DeWalt, Milwaukee）的颜色搭配和外形。
4. **硬度造假**：手动工具标注CR-V（铬钒钢）但实际使用碳钢，会导致买家退货并索赔。
5. **电机虚标**：扭矩（Nm）和功率虚标是国际站投诉重灾区，需提供真实测试报告。
6. **生锈问题**：五金件海运周期长，防锈油涂抹不足或包装防潮不到位会导致到货生锈。
7. **重量差异**：五金件运费占比较高，报价时务必精确计算整箱重量，避免运费亏损。
8. **售后配件**：锂电产品需承诺一定比例的电池、充电器配件供应，否则买家难以建立长期信任。
