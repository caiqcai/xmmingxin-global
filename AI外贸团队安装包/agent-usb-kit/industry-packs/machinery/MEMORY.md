# 行业知识 — 机械设备（Machinery）

## 核心品类方向
### 主力品类
- **农业机械 (Agricultural Machinery)**：
  - 小型拖拉机 (Compact Tractors)：4WD、多功能适配（犁、播种、除草），中低马力需求大。
  - 智能无人喷药机 (Ag-Drones)：侧重多点避障、大容量药箱及RTK精准定位。
  - 饲料机械 (Feed Pellet Mills)：家禽/家畜饲料加工，侧重压轮耐磨性与产量。
- **激光及自动化 (Laser & Automation)**：
  - 光纤激光切割机 (Fiber Laser Cutters)：1000W-12000W功率，侧重床身稳定性及控制系统兼容性。
  - 激光焊接机 (Laser Welders)：手持式设计，侧重轻便化、连续作业时间及焊缝美观度。
  - 激光打标机 (Laser Markers)：CO2/紫外/光纤打标，侧重打标深度与速度。
- **包装机械 (Packaging Machinery)**：
  - 真空包装机 (Vacuum Packers)：家用及商用，侧重抽气速率与封口宽度。
  - 自动贴标机 (Labeling Machines)：平面/圆瓶贴标，侧重定位精度及速度。
  - 液体/粉末灌装线 (Filling Lines)：针对化妆品、食品、医药行业，侧重防漏设计与易清洗。
- **工程机械 (Construction Machinery)**：
  - 小型挖掘机 (Mini Excavators)：0.8吨-3吨级，城市基建、园林绿化及农场作业利器。
  - 滑移装载机 (Skid Steer Loaders)：侧重多属具切换能力（铲斗、叉车、扫雪器）。
  - 混凝土机械 (Concrete Machines)：小型搅拌机、喷浆机，侧重泵送高度与耐用性。
- **食品加工机械 (Food Processing)**：
  - 商业制冰机 (Ice Makers)：方冰/雪花冰，侧重日产量及储冰槽保温效果。
  - 自动面条机/压面机：面馆、中央厨房配套，侧重面带均匀度。
  - 榨油机 (Oil Press)：侧重出油率、多功能压榨（大豆、花生、向日葵）。
- **木工机械 (Woodworking Machinery)**：
  - 数控开料机 (CNC Routers)：针对家具定制，侧重加工中心精度及吸附台面。
  - 封边机 (Edge Banders)：全自动/半自动，侧重封边美观度与胶合强度。

### 新兴/蓝海方向
- **电动化机械 (Electric/Li-ion Machinery)**：如锂电微型挖掘机、电动叉车，响应全球绿色能源及环保减排趋势。
- **协同机器人 (Cobots)**：轻型机械臂在中小工厂装配、码垛、喷涂场景的应用。
- **3D打印机械 (Industrial 3D Printers)**：针对模具、原型制作及复杂零件生产的大尺寸工业打印设备。
- **实验室/医疗自动化设备**：针对高精度液体处理、自动离心等场景的实验室自动化。

## 关键认证与合规
| 目标市场 | 必要认证 | 说明 |
|----------|---------|------|
| 欧盟 | CE (MD/LVD/EMC) | 机械指令(MD)是强制性安全标准，涉及急停、防护罩、联锁等安全设计 |
| 美国 | EPA / CARB | 凡涉及内燃机、柴油机的机械（如挖机、发电机），必须符合联邦或加州的排放标准 |
| 全球 | ISO 9001 | 基础制造体系认证，是大宗采购及政府招标项目的入门条件 |
| 俄罗斯/独联体 | EAC | 欧亚联盟强制性认证，通关必备，对标签、手册、合格证明有严格要求 |
| 全球 | RoHS / REACH | 涉及机械电气部分的材料环保合规，主要是针对出口至欧盟的设备 |
| 澳洲 | AS/NZS Standards | 针对特定的起重、压力容器及电气机械，需符合澳大利亚/新西兰本地标准 |

## 旺季节奏
- **3-4月 (Spring Kick-off)**：北半球开春，农业机械进入高峰；全球基建工程复工，小型挖机、装载机需求猛增。
- **5-7月**：工业包装及贴标机械需求平稳，制冰机、榨汁机等食品机械旺季。
- **9-11月 (Industrial Update)**：9月采购节爆发，大型激光设备、数控机床及生产线更新换代的高峰期。
- **12月**：年底询盘潮，主要针对明年初的定制产线和预算外采购。

## 营销与运营建议
- **视频化表达**：机械类详情页必须包含“工厂实景+空载试机+带料试机+细节展示”的四维视频体系。
- **参数矩阵**：详情页第一屏必须提供清晰的“技术规格表”，重点标注功率、效率、重量及核心件品牌（如Yaskawa, IPG）。
- **专业询盘回复**：提供“解决方案式”回复。例如：买家咨询切管机，回复中应附带不同厚度管材的切割样片及耗时数据。
- **零部件超市**：在店铺中专门设置“Spare Parts”分类，解决买家对售后维护的后顾之忧。

## 行业避坑指南
1. **电压/插头适配死穴**：全球工业/民用电标准差异极大（110V-440V, 50/60Hz），必须通过PI书面确认并保留证据，否则退运成本极高。
2. **海运包装与防锈**：长达30-60天的海上运输，盐雾腐蚀极其严重。机械必须涂防锈油并采用真空包装+铝箔袋+出口级免熏蒸木箱。
3. **手册与标识的本地化**：所有按钮、标签、系统界面、操作说明书必须为标准英文（或当地语种）。缺少手册或标签错误会导致当地清关受阻及买家拒绝签收。
4. **易损件的“长尾”坑**：机械必定涉及耗材（如刀片、密封圈、滤芯）。如果不随货赠送第一套易损件或拒绝销售配件，买家会认为供应商不专业。
5. **重型货物运费陷阱**：海运费波动及港口杂费（THC, DDC）远超买家预期。报价必须明确成交术语（FOB/CIF/DDP），特别是DDP报价需明确不包含哪些特定税费。
6. **安装调试与远程支持**：大型设备必须有详细的安装指导视频。若不具备上门安装能力，必须提前说明，并在详情页展示远程视频指导服务的案例。
7. **认证虚标或过期**：CE证书是否在欧盟公告机构(NB)库中可查是决定因素。虚标证书会导致海关直接没收机器并对卖家进行处罚。
8. **忽略保修期限起始点**：必须明确保修期是从“出厂日”还是“签收日”开始。建议从签收日开始并提供延保方案增加转化。
