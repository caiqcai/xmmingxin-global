# 行业知识 — 家具（Furniture）

## 核心品类方向
### 主力品类
- **办公家具（Office Furniture）**：人体工学椅、升降桌、文件柜，B2B需求量最大。
- **客厅家具（Living Room Furniture）**：沙发、电视柜、茶几，体积大，对物流依赖度高。
- **卧室家具（Bedroom Furniture）**：床架、床垫、梳妆台，重点在于环保等级和设计感。
- **餐厅家具（Dining Room Furniture）**：餐桌、餐椅、吧台椅，金属与木材结合类产品热销。
- **户外家具（Outdoor Furniture）**：藤编沙发、遮阳伞、铝合金座椅，极具季节性。
- **酒店/商用家具（Hospitality Furniture）**：定制化程度高，利润空间大。

### 新兴/蓝海方向
- **智能升降办公系统（Smart Standing Desks）**：带记忆功能、无线充电、线缆管理的升降桌。
- **环保可持续家具（Eco-friendly Furniture）**：使用FSC认证木材、可回收金属或海洋塑料制成的家具。
- **多功能小户型家具（Space-saving Furniture）**：折叠沙发床、隐藏式收纳家具，适配城市化进程。

## 关键认证与合规
| 目标市场 | 必要认证 | 说明 |
|----------|---------|------|
| 全球 | FSC | 森林认证，木材家具进入欧美主流市场的“身份证” |
| 美国 | CARB / EPA | 甲醛释放量限制，极其严格，违规会被海关扣押 |
| 美国 | BIFMA | 办公家具（尤其是椅子）的耐用性测试标准 |
| 英国 | BS 5852 | 英国阻燃测试标准，沙发及软体家具必测 |

## 旺季节奏
- **1月-3月（开年采购）**：针对春季装修市场及新办公楼入驻。
- **8月-10月（下半年旺季）**：九月采购节，准备圣诞及新年家庭焕新。
- **5月-7月**：户外家具的出货高峰（需提前3个月下单）。

## 行业避坑指南
1. **体积与包装**：家具多为抛货，务必推行KD（Knock Down，拆装）包装以降低运费。
2. **说明书陷阱**：组装类家具若说明书不清晰，会导致大量售后纠纷及退款。
3. **甲醛风险**：低价板材家具容易甲醛超标，必须要求工厂提供板材环保测试报告。
4. **物流损坏**：大理石面板、玻璃件必须木架包装，否则破损率高达15%以上。
5. **承重虚标**：椅子、床架的静态/动态承重需有实验室报告，断裂导致的受伤赔偿极高。
6. **五金件缺失**：螺丝、扳手等五金包在打包环节极易漏装，需二次复核。
7. **色差问题**：面料（丝绒、皮料）在不同光线下色差大，必须在描述中声明或寄送色卡。
8. **清关合规**：美国对实木家具有反倾销税（部分品类），发货前务必查询最新税率。
