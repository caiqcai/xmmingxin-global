# 行业知识 — 加工服务（Fabrication Services）

## 核心品类方向
### 主力品类
- **CNC机加工（CNC Machining）**：车削、铣削、高精度零件加工，B端需求量最稳。
- **钣金加工（Sheet Metal Fabrication）**：激光切割、折弯、焊接，针对机箱机柜、结构件。
- **注塑模具/加工（Injection Molding）**：塑料件开模及批量生产，属于高黏度订单。
- **3D打印服务（3D Printing）**：SLA/SLM（金属/树脂）快速成型，主要针对研发初期。
- **铸造与锻造（Casting & forging）**：针对重型机械、泵阀零件，大宗工业订单。
- **PCB/PCBA加工（Electronics Contract Manufacturing）**：电子代工，从制板到贴片。

### 新兴/蓝海方向
- **一站式原型制作（24-Hour Rapid Prototyping）**：针对硬件创业公司，强调交付速度而非单价。
- **复杂金属3D打印（Metal Additive Manufacturing）**：用于传统切削无法实现的精密中空结构件。
- **小批量定制化（Low-volume Production）**：10-100件起订，适配个性化、碎片化的现代供应链需求。

## 旺季节奏
- **3月/9月**：国际站大促期，适合获取新客户询盘。
- **10月-12月**：针对下一年度新产品的研发打样高峰。
- **全年**：加工服务无明显季节性，受新产品发布周期驱动。

## 行业避坑指南
1. **图纸版本管理**：务必确认买家提供的图纸是最终版本（Final Revision），避免按旧图加工。
2. **公差陷阱**：未注公差默认按ISO 2768级处理，必须在报价前与工程师确认关键尺寸公差。
3. **表面处理色差**：阳极氧化（Anodizing）存在缸差，大批量件需声明轻微颜色不均。
4. **材料证明（MTR）**：买家常要求材料成分分析报告（Material Test Report），若材料不达标（如假6061），会导致退整批货。
5. **知识产权（IP）保护**：签署NDA（保密协议）是常态，泄露买家图纸会导致法律诉讼。
6. **包装设计**：精密零件易碰伤、刮花，需使用气泡袋隔开并装入珍珠棉定制格栅箱。
7. **计费逻辑**：明确是含料加工还是来料加工，通常包含首次打样费和后续量产单价。
8. **跨国沟通障碍**：加工行业术语多，务必通过标注清晰的PDF图纸沟通，而非口头确认。
