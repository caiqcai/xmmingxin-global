# 行业知识 — 安防（Security & Protection）

## 核心品类方向
### 主力品类
- **闭路电视/监控（CCTV/Surveillance）**：IP摄像机、NVR套装、智能猫眼，AI识别功能是竞争核心。
- **个人防护装备（PPE）**：安全鞋、工作服、反光背心、防尘口罩，刚需消耗品。
- **门禁控制（Access Control）**：智能锁、门禁卡、对讲系统，Tuya/TTlock生态需求旺盛。
- **消防安全（Fire Safety）**：烟雾报警器、灭火器、消防毯，属于强制合规类产品。
- **道路安全（Road Safety）**：路锥、减速带、交通指示牌，多为政府或工程项目采购。
- **警用/战术装备（Police & Tactical）**：战术背心、对讲机附件、手电筒，特定买家群体。

### 新兴/蓝海方向
- **AI边缘计算摄像机（AI Edge Cameras）**：内置人脸识别、人形检测、车牌识别，无需依赖云端处理。
- **智能家庭报警系统（DIY Home Security Kit）**：全套无线传感器、警报器，支持手机APP一键操控。
- **时尚化PPE（Fashionable PPE）**：像运动鞋一样的安全鞋、带设计感的工作服，受年轻工人欢迎。

## 关键认证与合规
| 目标市场 | 必要认证 | 说明 |
|----------|---------|------|
| 欧盟 | CE (EN20345) | 安全鞋的强制标准，涉及S1/S1P/S3等不同防护等级 |
| 美国 | FCC / UL | 电子类安防产品的射频及电气安全强制认证 |
| 全球 | RoHS | 涉及电子产品的有害物质检测 |
| 欧盟 | EN 14604 | 烟雾报警器的欧洲准入标准，极其严格 |

## 旺季节奏
- **3月/9月采购节**：全年订单高峰。
- **4月-6月**：户外监控及道路安全产品的安装旺季。
- **11月-1月**：冬季室内防火安全及家庭安防产品的需求上升。

## 行业避坑指南
1. **APP服务器稳定性**：安防电子产品的APP如果海外访问缓慢，会产生大量退款和差评。
2. **PPE防护等级虚标**：安全鞋的钢头耐冲击力若达不到标准，在欧美发生工伤事故会导致巨额索赔。
3. **隐私合规**：摄像机必须符合GDPR（欧洲数据保护条例），严禁默认密码不强制修改。
4. **消防取证极其困难**：烟雾报警器等产品没有VdS或UL认证，千万不要强行发货到欧美，极易被没收。
5. **海运电池问题**：摄像头、智能锁多带电池，需确认为可拆卸或提供MSDS随货。
6. **外观侵权**：大厂（如海康、大华）的外观专利非常多，公模产品需确认不侵权。
7. **说明书多语言**：安防系统设置复杂，必须提供精准的英语、德语、法语等多语言说明书。
8. **清关名义**：部分国家对“监控设备”有特殊进口许可要求，报价前需确认买家清关能力。
