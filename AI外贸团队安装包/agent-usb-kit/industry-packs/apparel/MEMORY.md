# 行业知识 — 服装（Apparel）

## 核心品类方向
### 主力品类
- **女装连衣裙 (Women's Dresses)**：
  - 碎花裙 (Floral Dresses)：波西米亚风格、春夏季热销，侧重雪纺和真丝替代面料。
  - 晚礼服 (Evening Gowns)：针对舞会、婚礼季，侧重蕾丝、亮片及修身剪裁。
  - 休闲日常裙 (Casual Dresses)：棉麻材质、大廓形设计，主打舒适与简约。
- **运动健身服 (Yoga & Activewear)**：
  - 高弹无缝瑜伽裤 (Seamless Leggings)：高腰提臀设计，主流需求集中在吸湿排汗与高弹包裹感。
  - 运动内衣 (Sports Bras)：中高强度支撑，防震设计，侧重透气网眼应用。
  - 速干衣 (Dry-fit Sets)：跑步及综合训练服，强调面料的轻盈与速干性能。
- **卫衣与连帽衫 (Hoodies & Sweatshirts)**：
  - 重磅卫衣 (Heavyweight Hoodies)：350g-500g纯棉面料，廓形(Oversized)剪裁，Z世代核心单品。
  - 定制印花卫衣 (Custom Graphic Hoodies)：发泡印花(Puff Print)、刺绣工艺的应用。
- **T恤 (T-Shirts)**：
  - 基础款 T 恤：高支棉(High-count Cotton)、抗皱免烫工艺，侧重面料触感。
  - 复古文化衫 (Vintage Tees)：水洗旧化工艺、大幅背部印花设计。
- **泳装 (Swimwear)**：
  - 比基尼 (Bikinis)：多绳结设计、独特纹理面料（如提花、褶皱）。
  - 冲浪服与防晒衣：功能性防护(UPF 50+)，侧重速干与耐氯性能。
- **外套与夹克 (Jackets & Coats)**：
  - 轻便羽绒服 (Lightweight Puffer)：侧重可收纳性及防水涂层。
  - 棒球夹克 (Varsity Jackets)：拼色设计、贴布绣工艺，校园风长盛不衰。

### 新兴/蓝海方向
- **可持续/环保服装 (Sustainable Apparel)**：使用再生聚酯(rPET)、有机棉、竹纤维、汉麻纤维面料，满足欧美ESG采购要求。
- **无性别/中性风 (Unisex/Gender-neutral Fashion)**：模糊性别的剪裁设计，同一版型适配男女买家，降低SKU压力。
- **智能温控服装 (Smart Tech Apparel)**：集成电加热模块、智能变色纤维的户外功能服。
- **大码时尚 (Plus Size Inclusive)**：北美及中东市场的高增速蓝海，侧重剪裁对体型的修饰与包容。

## 买家画像
| 维度 | 描述 |
|------|------|
| 主力采购市场 | 北美（美国、加拿大）、欧洲（英、德、法、意）、大洋洲（澳洲）、中东（沙特、阿联酋） |
| 典型买家类型 | 独立站(Shopify)时尚店主、精品买手店(Boutique)老板、亚马逊Top服装卖家、成熟时尚品牌商、社交媒体网红自创品牌 |
| 采购频次与周期 | 季节性极强；快时尚买家每周关注新品并小批量试单；普通品牌每季度大批量备货一次 |
| 单次采购量级 | 现货(RTS)试单3-20件；定制订单(OEM/ODM)起订量通常在50-300件/色/款 |
| 决策关注点 | 设计的独特性、面料的手感与耐洗度、尺码表的标准性、打样及交期速度、定制工艺多样性 |
| 价格敏感度 | 中等；买家更愿意为优秀的原创设计、高品质面料和灵活的定制方案（小MOQ定制）支付溢价 |

## 关键认证与合规
| 目标市场 | 必要认证 | 说明 |
|----------|---------|------|
| 全球 | OEKO-TEX Standard 100 | 纺织品生态安全认证，证明不含对人体有害的化学物质 |
| 全球 | GRS (Global Recycled Standard) | 全球回收标准，环保回收类产品必须提供的链路证明 |
| 全球 | BSCI / Sedex / WRAP | 社会责任审核，进入欧美大型连锁零售渠道及中高端品牌的门票 |
| 美国 | CPSC (CPSIA) | 尤其是童装，必须符合铅、邻苯二甲酸盐限量及阻燃性标准 |
| 欧盟 | REACH | 化妆品及纺织品涉及的化学成分合规，严禁特定偶氮染料 |
| 全球 | Organic Content Standard (OCS) | 有机棉含量认证，用于支撑“Organic”营销卖点 |

## 旺季节奏
- **1-3月 (Spring/Summer Kick-off)**：春夏季货品备货期，3月新贸节是女装全年成交最高峰。
- **4-5月**：泳装、防晒衣及度假裙装的补货高峰。
- **7-9月 (Autumn/Winter Peak)**：秋冬季大宗备货期，9月采购节爆发，卫衣、夹克类需求猛增。
- **10-11月 (Holiday Prep)**：圣诞、派对裙装及滑雪装备的最后补货窗口。
- **12月**：年底清仓及次年春装打样、设计趋势确认期。

## 营销与运营建议
- **实拍视频**：服装类买家对“实物感”要求极高，主图必须包含面料细节微距视频及模特走位视频。
- **尺码表优化**：提供厘米(cm)和英寸(inch)双语尺码表，并明确区分亚洲码、欧洲码及美码。
- **小起订量(Low MOQ)定制**：提供“50件起印Logos”的柔性服务是转化Shopify店主的核心竞争力。
- **网红营销素材**：提供高质量的无水印白底图及INS风买家秀图，方便B端买家直接用于其独立站推广。

## 行业避坑指南
1. **尺码退货风险**：亚洲码通常比美码小两号，未明确标注会导致极高的退货率和差评。
2. **色差投诉**：显示器与实际布料色差是永恒的痛点，建议在详情页声明“自然光下拍摄，可能存在微弱色差”。
3. **面料成分真实性**：化纤冒充棉、普通聚酯冒充再生聚酯在海关抽查或买家实验室测试中风险极大。
4. **版权法律陷阱**：严禁直接采集海外快时尚网站、INS网红图，一旦被权利人投诉，将面临平台封店及高额索赔。
5. **季节性滞销**：服装生命周期短，生产或物流延误20天可能导致该批次货品错过当地最佳销售期。
6. **异味与线头**：新面料的甲醛味或包装袋产生的异味会严重影响买家首印象，出货前需进行充分通风及剪线头质检。
7. **物流包装优化**：服装是耐挤压品，应尽可能使用真空包装减少体积，降低运费成本，增加竞争力。
8. **印花耐久性**：低成本丝网印花易开裂，出口订单建议使用热转印或高品质数码直喷，并告知洗涤说明。
