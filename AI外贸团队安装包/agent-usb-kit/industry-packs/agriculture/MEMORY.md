# 行业知识 — 农业（Agriculture）

## 核心品类方向
### 主力品类
- **农业机械 (Agricultural Machinery)**：轮式/履带拖拉机（25HP-200HP）、小型微耕机、水稻插秧机、播种机、联合收割机、饲料粉碎机。这是农业出海单价最高、需求最稳的核心大项。
- **灌溉系统 (Irrigation Systems)**：PE/PVC滴灌带、微喷头、喷灌机组、太阳能水泵、自动灌溉施肥一体机。全球气候干旱背景下，节水农业是多国政府扶持的重点。
- **农业温室与园艺设施 (Greenhouse & Horticulture)**：多跨连栋温室、塑料薄膜温室、水培（Hydroponic）系统、人工补光灯（Grow Lights）、温室自动温控卷帘。
- **农化产品 (Agrochemicals)**：水溶性化肥（NPK）、有机肥、生物刺激素、环保型除草剂、杀虫剂及植物生长调节剂。
- **动物饲料及添加剂 (Animal Feed & Additives)**：鱼粉、大豆浓缩蛋白、氨基酸添加剂、益生菌、磷酸二氢钙。针对养殖业的工业化升级。
- **畜牧业设备 (Animal Husbandry Equipment)**：自动化投料机、饮水碗、挤奶机、孵化器、牲畜耳标。
- **农产品初加工设备 (Post-harvest Equipment)**：磨粉机、碾米机、茶叶烘干机、坚果脱壳机、果蔬冷藏保鲜设备。针对非洲及东南亚农产品附加值提升的需求。
- **园艺工具 (Garden Tools)**：电动修枝剪、割草机、树篱剪、园艺铲、播种盘、防虫网。

### 新兴/蓝海方向
- **智能植保无人机 (Agricultural Drones)**：集成GPS定位与自动避障，用于大规模药剂喷洒与农田监测。由于人工成本上升，在拉美、东南亚及中亚地区正呈爆发式增长。
- **室内垂直农场与植物工厂 (Vertical Farming)**：模组化LED垂直种植货架、营养液自动循环系统。针对城市农业、沙漠地区及极寒地带的特殊种植解决方案。
- **生物有机农业投入品 (Bio-organic Solutions)**：海藻精、腐植酸、拮抗细菌等生物农药。随着全球有机食品标准收紧，传统化学农药的替代品利润极高。
- **智慧农业监测传感器 (Smart Farming Sensors)**：土壤温湿度传感器、EC值检测仪、农业气象站。针对精准农业的数据采集层。

## 买家画像
| 维度 | 描述 |
|------|------|
| 主力采购市场 | **非洲/东南亚**（低成本农机、初加工、灌溉）、**拉美**（大中型农机、植保无人机、农化）、**中东**（沙漠节水灌溉、高端温室）、**欧/美/澳**（园艺工具、垂直农业、生物农药） |
| 典型买家类型 | **地方级农资/农机代理商**、**大型种植园/养殖场主**、**政府农业部/非政府组织 (NGO) 采购方**、**园艺连锁零售店**、**农业科技创业公司**、**农业合作社** |
| 采购频次与周期 | **强季节性**。播种季节前3-6个月是采购巅峰；农机具通常3-5年更新一次，但配件（如犁铧、皮带）是年年采购。 |
| 单次采购量级 | **农机**：1-10台/单；**农资耗材**：10吨-100吨（化肥/饲料）；**园艺工具**：500-2000件；**温室工程**：按面积计，通常1000平米起订。 |
| 决策关注点 | **当地适应性**（土壤硬度/气候）、**操作与维修简易度**、**零配件供应能力**、**技术指导（SOP）**、**政府补贴资质**。 |
| 价格敏感度 | **极高**。农业是微利行业，买家极度看重投资回收期。但对于高科技设备（如无人机），买家也看重售后培训与维护网点。 |

## 关键认证与合规
| 目标市场 | 必要认证 | 说明 |
|----------|---------|------|
| 欧盟 | CE / RoHS | 所有的动力农机、电子设备必须符合CE安全指令，农残标准符合RoHS。 |
| 美国 | EPA / CARB | 凡涉及汽油/柴油发动机的农机（如拖拉机、发电机），必须符合美环保署排放标准。 |
| 全球 | 植物检疫证书 (Phytosanitary) | 涉及种子、种苗、木制包装或部分有机肥料，出口必须有政府签发的植检证。 |
| 全球 | MSDS / COA | 化肥、农药等化工品必须提供安全技术说明书（MSDS）及成分分析报告（COA）。 |
| 东南亚/非洲 | PVOC / SONCAP | 肯尼亚、尼日利亚等国对入关产品有强制的前期装船前检验要求。 |

## 旺季节奏
- **1月-3月**：**北半球春耕蓄力期**。化肥、种苗盘、灌溉管网、小型整地机进入订单疯狂期。买家必须在播种前收到货。
- **4月-6月**：**管理与园艺季**。除草机、植保无人机、水泵、园艺修剪工具需求达到巅峰。
- **7月-9月**：**南半球备货与收割季**。巴西、澳洲等南半球大农场主开始采购下半年播种物资；北半球收割机配件需求激增。
- **10月-12月**：**温室建设与畜牧储备期**。寒冷地区温室工程启动；畜牧业为度冬进行饲料及保暖设备（如加热灯）的大宗采购。

## 行业避坑指南
1. **忽略排量与排放限制（“农机废铁”）**：拖拉机发到美国却不符合EPA，或发到欧洲不符合Euro 5。海关会直接扣押并责令退运或现场销毁，卖家损失惨重。
2. **液体/粉末农资的包装灾难**：化肥多为粉末或强酸碱液体，属于敏感化工品。包装不牢会导致漏液、吸潮结块，严重时会导致整柜腐蚀。必须使用多层加厚内衬袋及防溢阀包装。
3. **忽略零配件的连续性**：买家在偏远农场，最怕机器坏了没零件。若随货不带易损件包（Filter, Belts, Spark plugs），买家一次售后无果后会给致命差评。
4. **忽视清关单证的“罗生门”**：不同国家对农药/化肥的植检证（Phyto）、成分证明（COA）、产地证（CO）格式要求极其死板。错一个公章就无法入境。
5. **电压、频率与燃油质量**：温室控制系统需确认电压（110V/220V/380V）；发电机需确认当地燃油辛烷值/十六烷值是否适配，否则发动机会严重积碳甚至报废。
6. **对易碎品（阳光板/玻璃）保护不足**：温室建材体积大且脆。必须采用专业A型木架支撑，并购买高额保险，否则破损是家常便饭。
7. **低估农业季节性的延误风险**：错过一周播种期，买家一年没收成。由于物流延误导致的错失农时，买家弃货率是100%。
8. **农机外观侵权**：部分国产农机外观仿造John Deere（强鹿）或Kubota（久保田）。这些大牌在多国都有外观专利，极易被告侵权导致港口扣货。
9. **忽略当地土壤特性**：同样是犁，在非洲红土地和拉美黑土地的磨损率完全不同。不了解买家土地情况直接推荐，易造成产品寿命极短的质量纠纷。
