# 行业知识 — 冶金矿产（Minerals & Metallurgy）

## 核心品类方向
### 主力品类
- **铝型材及铝制品 (Aluminum Profiles)**：门窗型材、工业铝挤压材、太阳能组件支架、铝板、铝箔。由于轻量化及耐腐蚀特性，在建筑及工业领域需求极其旺盛。
- **不锈钢卷与板 (Stainless Steel Coils/Sheets)**：304、316、201、430等系列。涵盖镜面板、拉丝板、彩色不锈钢。广泛用于厨具、装饰、医疗及重工业。
- **铜及其合金 (Copper & Alloys)**：纯铜管、黄铜棒、紫铜箔、漆包线。主要应用于电力传输、制冷工程及精密电子元器件。
- **钢管与管件 (Steel Pipes & Fittings)**：无缝钢管、螺旋管、镀锌管、法兰、弯头。是全球石油、化工、水利及建筑工程的骨架材料。
- **特种合金与稀有金属 (Specialty Alloys & Rare Metals)**：镍基合金、钛合金板、钨钢圆棒、钼制品。这些材料多用于航空航天、核电及高性能切削工具。
- **非金属矿产 (Non-metallic Minerals)**：石墨粉（电池级/工业级）、膨润土、高岭土、蛭石、石英砂。作为工业添加剂或耐火材料，走量巨大。
- **耐火材料 (Refractory Materials)**：耐火砖、浇注料、陶瓷纤维毯。主要销往玻璃厂、水泥厂及冶金厂的炉窑改造项目。
- **金属丝网 (Metal Mesh)**：不锈钢网、围栏网、冲孔网、扩充网。用于建筑过滤、安防及装饰。

### 新兴/蓝海方向
- **低碳/绿色金属材料 (Green Metals)**：符合全球ESG（环境、社会和治理）标准的“绿铝”（水电铝）、“低碳钢”。欧美高端制造企业为达成减碳目标，对这类产品有明显的采购溢价。
- **新能源电池关键矿物 (Battery Minerals)**：高纯度硫酸镍、硫酸锰、人造石墨负极材料、碳纳米管导电剂。受益于全球电动汽车产业链爆发。
- **3D打印金属粉末 (Metal Powders for AM)**：钛合金粉、高温合金粉、不锈钢粉末。随着工业3D打印（增材制造）的普及，高毛利的金属粉末市场增速飞快。
- **高性能复合材料 (High-performance Composites)**：碳纤维增强复合材料（CFRP）及其配件，正在逐步替代部分传统的金属结构件。

## 买家画像
| 维度 | 描述 |
|------|------|
| 主力采购市场 | **东南亚**（基建与制造移入）、**中东**（油气基建与城市亮化）、**欧洲/北美**（高端制造、实验室耗材、绿色能源配件）、**拉美**（矿山设备及基础建材） |
| 典型买家类型 | **中小型制造企业 (SMEs)**（需原材料加工）、**建筑承包商/工程商**、**大宗原材料贸易商/分销商**、**工业超市/批发商**、**科研院校实验室** |
| 采购频次与周期 | **周期性极强**。贸易商通常根据库存水平按月补货（1个月左右）；大型项目方则按工程进度大批量一次性采购。 |
| 单次采购量级 | **大宗商品**：10吨-100吨（FCL整柜）是常态；**特种金属**：100kg-500kg起订；**非金属矿产**：通常25kg/袋，500袋起订。 |
| 决策关注点 | **材质证明单 (MTC)**（绝对核心）、**理化指标一致性**、**公差控制精度**、**供货稳定性与账期**、**包装防锈能力**。 |
| 价格敏感度 | **极高**。大宗原材料属于低利润率、高周转行业。买家对LME（伦敦金属交易所）价格及海运费波动极其敏感，毫厘之差决定成交。 |

## 关键认证与合规
| 目标市场 | 必要认证 | 说明 |
|----------|---------|------|
| 全球 | ISO 9001 / ISO 14001 | 基础质量与环境体系证明，是进入大型跨国公司供应商库的敲门砖。 |
| 全球 | SGS / BV / TUV 第三方化验报告 | 对产品成分（如不锈钢中的Cr, Ni含量）进行第三方抽检，是买卖双方信任的基础。 |
| 欧盟 | CE (EN 1090) | 结构钢、铝型材进入欧盟建筑市场必须符合CPR指令及EN 1090执行标准。 |
| 美国 | ASTM / ASME / AISI | 美国材料试验学会等标准，不符合标准则无法进入美国主流工程及军工民用市场。 |
| 印度 | BIS | 钢铁及部分有色金属出口印度必须申请BIS认证，审核流程极长且严苛。 |

### 避坑词（5-8个需要避免的词）
- **Cheapest** (低价词虽然流量大，但转化出的多是劣质客户，纠纷多)
- **100% Pure** (工业上很难达到绝对100%，建议写 99.9% 或 99.99%，避免抽检纠纷)
- **Best Seller** (滥用此类主观词易被系统判定为夸大宣传并降权)
- **Scrap Metal** (废金属贸易涉及极其复杂的进出口管制及环保限制，建议避免)
- **Rare Earths** (直接以此词发品常触动敏感物资监控，建议用具体品名如 Neodymium)
- **Chemical Weapons Precursor** (绝对红线词，禁售)
- **Gold/Diamond** (此类高价值矿产在国际站有极严的准入及资质限制)

## 旺季节奏
- **3月-5月**：**春季开工采购潮**。全球特别是北半球建筑、基建、电力工程进入全面开工季，对铝型材、钢管、电缆材料需求达到全年最高峰。
- **6月-8月**：**夏日维护与电力季**。工厂炉窑维护（耐火材料需求）、空调制冷用铜管、工业气体管道配件需求稳健。
- **9月-11月**：**“金九银十”年终冲刺**。项目方为了在年底前完工并结算，会进行最后一波大规模采购。
- **12月-2月**：**库存策略期**。欧美买家在12月处于休假清库，但东南亚及中国春节前的备货需求会支撑市场，卖家需注意假期工厂停工风险。

## 行业避坑指南
1. **成分造假（“货不对板”）**：这是冶金行业最大的死穴。用201不锈钢冒充304，或者用低厚度铝材冒充标厚。由于大多需要专业光谱仪或化验室检测，买家收货后一旦化验不符，索赔往往是整单全赔加巨额运费。
2. **防锈包装不到位**：金属材料跨海运输面临严重的盐雾腐蚀。若不使用优质防锈油、VCI防锈袋及加厚编织带，并配合木托盘加固，到港时生锈的钢材会被买家弃货。
3. **反倾销税与地缘政策**：铝制品、钢制品在许多国家（如美国、印度、欧洲）面临极高的反倾销税（甚至高达100%+）。发货前务必查询该HS编码在买家国家的关税政策，否则货物必死无疑。
4. **MTC（材质单）的缺失或造假**：正规买家必看MTC（Material Test Certificate）。若卖家无法提供，或提供的证书与实际货物批次不符，将被视为商业欺诈。
5. **装箱不当导致爆柜**：金属制品极重。若集装箱内重量分布不均或未进行有效的木架加固，在海运颠簸中会导致集装箱受力不均损坏，甚至导致买家在清关时因安全问题被罚。
6. **忽略价格波动的合同陷阱**：大宗商品价格波动快。在PI（形式发票）中务必注明：Quote Valid for 3 Days，且必须以款到账时的原材料行情为准，否则一次原材料暴涨就足以让卖家亏损倒闭。
7. **HS编码的“迷宫”**：金属制品的HS编码归类极细（如管件、板材、型材），归类错误不仅涉及退税损失，还可能让买家因“错报税号”而面临海关调查。
8. **忽略特定国家出口管制**：部分高性能特种合金、高纯度石墨涉及出口许可证管理，未办妥证件强行出口会导致货物被查封及退运。
9. **证书名称的主体关联性**：买家清关时，材质证书、CE证书上的工厂名称必须能与出口商（你）建立关联（如授权书），否则会被拒。
