# 行业知识 — 珠宝饰品（Jewelry & Accessories）

## 核心品类方向
### 主力品类
- **Fashion Jewelry (流行饰品)**: 合金、铜底、亚克力等低单价、快时尚产品。
- **Sterling Silver Jewelry (925银饰)**: 包含项链、戒指、耳环，最主流的中端品类。
- **Body Jewelry (人体穿刺饰品)**: 鼻环、唇钉、腹部环，针对细分硬核市场。
- **Mens Jewelry (男士饰品)**: 钛钢链条、印章戒指，近年来增长迅猛。
- **Eyewear & Sunglasses (眼镜/墨镜)**: 季节性强，重视防紫外线功能与时尚度。
- **Watches (手表)**: 包含石英表、智能手表及表带配件。

### 新兴/蓝海方向
- **Sustainable/Lab-Grown Diamonds**: 培育钻石产品，针对婚庆及高端日常市场。
- **Spiritual/Meaningful Jewelry**: 包含星座、疗愈水晶、好运符号等具有情感价值的饰品。
- **PVD Gold Plated / Waterproof Jewelry**: 采用高品质电镀技术，主打不掉色、可佩戴洗澡，深受欧美买家喜爱。

## 关键认证与合规
| 目标市场 | 必要认证 | 说明 |
|----------|---------|------|
| 全球 | ISO 9001 | 基础质量体系。 |
| 欧美 | REACH | 针对镍 (Nickel)、铅 (Lead)、镉 (Cadmium) 的限量要求。 |
| 美国 | Proposition 65 | 加州针对饰品金属成分的合规声明。 |
| 全球 | SGS / TUV Report | 针对特定材质（如 S925, 钛钢）的纯度与成分分析报告。 |
| 高端市场 | RJC (Responsible Jewellery Council) | 责任珠宝业委员会认证，针对大品牌供应链。 |

## 发品关键词库

Jewelry Set, Necklace, Earring, Ring, Bracelet, Bangle, Sunglasses, Watch, Piercing Jewelry, 925 Sterling Silver, Stainless Steel Jewelry, Custom Jewelry.

### 长尾蓝海词（10-15个）

## 旺季节奏
- **1月-2月**: 情人节备货期及 3 月新贸节。
- **3月**: **新贸节**。春夏新品发布，大量测款。
- **5月**: 母亲节备货尾声。
- **9月**: **超级九月**。下半年圣诞、黑五备货的核心月。
- **11月**: 圣诞现货补货及元旦、次年情人节预热。

## 行业避坑指南
1. **过敏投诉**: 饰品与皮肤接触，一旦含镍超标导致过敏，买家投诉率极高且可能导致巨额赔偿。
2. **电镀掉色**: 低端水镀易掉色，若宣称不掉色却在短时间内氧化，将产生大量差评。
3. **侵权风暴**: 珠宝设计极其容易发生“外观专利”侵权，大牌的经典款（如四叶草、钉子）绝对不能碰。
4. **包装磨损**: 饰品精美但脆弱，发货需加固，避免运输途中碰撞导致掉石或断针。
5. **拍照欺骗**: 饰品通常很小，微距拍摄会让买家产生实物很大的错觉。主图必须包含“佩戴图”或“比例对照图”。
6. **真假乱码**: 如果是合成立方氧化锆（CZ），绝对不能直接写“Diamond”，必须注明。
