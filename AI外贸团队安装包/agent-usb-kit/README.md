# AI外贸团队 — U盘一键安装包

> 面向阿里巴巴国际站服务中客户，一键安装3个AI智能体。

## 团队阵容

| 角色 | 职责 | 技能数 |
|------|------|--------|
| 📊 **运营总监** | 管"店"：数据分析、SEO优化、流量运营、营销策划、统筹协调 | 16个 |
| 📦 **产品经理** | 管"货"：选品分析、1688找货、国际站发品、详情页优化 | 18个 |
| 🤝 **业务员** | 管"人"：客户开发、背调、邮件营销、询盘回复、商务谈判 | 16个 |

## 安装方式

### 对客户（使用方）

**U盘插上，双击就装，不用拷贝到电脑。**

| 系统 | 操作 |
|------|------|
| **Windows** | 插入U盘 → 打开U盘 → 双击「安装AI外贸团队.bat」→ 按提示填信息 → 完成 |
| **macOS** | 插入U盘 → 打开U盘 → 双击「安装AI外贸团队.command」→ 按提示填信息 → 完成 |

安装完成后：
1. 完全退出 Accio Work（macOS: Cmd+Q / Windows: 右键托盘退出）再重新打开
2. 侧边栏会出现3个新智能体
3. 把3个智能体拉成一个团队（推荐运营总监做队长）
4. 进入团队群打个招呼，队长会引导您上手

### 对销售（分发方）

1. 在 `agent-usb-kit` 目录下运行：`bash export.sh`
2. 生成的 zip 文件拷贝到 U 盘
3. 交给客户

## 安装流程

```
安装时收集（5项）：
  称呼 → 用于欢迎消息
  团队名 → 群组名称（默认：{称呼}的AI外贸团队）
  行业 → 36个类目选1，决定注入哪个行业知识包
  目标市场 → 定制关键词和提示词
  是否有团队 → 切换吉祥话版本

安装后自动完成：
  ✅ 创建3个智能体（产品经理/运营总监/业务员）
  ✅ 注入行业知识（MEMORY.md）
  ✅ 写入客户信息（USER.md）
  ✅ 生成首次对话引导（BOOTSTRAP.md）
  ✅ 设置像素风头像
  ✅ 模型设为Auto模式（自动省积分）
```

## 客户首次体验流程

```
手动组建团队（运营总监做队长）
  → 打开团队群
  → 运营总监（队长）自我介绍 + 介绍团队
  → 提醒绑定阿里国际站账号
  → 选择主营产品方向（3个热门品类+自定义）
  → 选一个服务体验团队协作
  → 引导开启定时任务（每日早报/每周选品/客户跟进）
```

## 技术说明

- 智能体ID格式：`DID-{6位随机HEX}-{32位随机HEX}`
- 安装零依赖：macOS 用纯 bash，Windows 用 PowerShell，不需要 python/node 等
- 模型默认Auto：`profile.jsonc` 中 `model: {provider: "auto", name: "auto"}`
