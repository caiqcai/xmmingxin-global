## 5. Google Rich Snippets: JSON-LD B2B Schema Template (结构化数据模板)

将以下 JSON-LD 代码粘贴到核心页面中，可以使您的网站在 Google 搜索结果中展示星级评分、服务类型等，极大地提高点击率：

### A. Homepage (或 About Page) 适用的 LocalBusiness & Brand Schema
```json
{
  "@context": "https://schema.org",
  "@type": "LocalBusiness",
  "name": "Your Custom B2B Manufacturing Brand",
  "image": "https://www.yourdomain.com/assets/images/factory-facade.jpg",
  "@id": "https://www.yourdomain.com/#localbusiness",
  "url": "https://www.yourdomain.com/",
  "telephone": "+86-XXXXXXXXXXX",
  "address": {
    "@type": "PostalAddress",
    "streetAddress": "No. XX Manufacturing Industrial Zone",
    "addressLocality": "Shenzhen / Ningbo",
    "addressRegion": "Guangdong / Zhejiang",
    "postalCode": "XXXXXX",
    "addressCountry": "CN"
  },
  "geo": {
    "@type": "GeoCoordinates",
    "latitude": 22.5431,
    "longitude": 114.0579
  },
  "openingHoursSpecification": {
    "@type": "OpeningHoursSpecification",
    "dayOfWeek": [
      "Monday",
      "Tuesday",
      "Wednesday",
      "Thursday",
      "Friday"
    ],
    "opens": "08:30",
    "closes": "18:00"
  },
  "sameAs": [
    "https://www.linkedin.com/company/your-manufacturing-page/"
  ]
}
```

### B. CNC Machining Service 页面适用的 Service Schema
```json
{
  "@context": "https://schema.org",
  "@type": "Service",
  "serviceType": "Precision CNC Machining Services",
  "provider": {
    "@type": "LocalBusiness",
    "name": "Your Custom B2B Manufacturing Brand",
    "url": "https://www.yourdomain.com/"
  },
  "areaServed": "Worldwide",
  "hasOfferCatalog": {
    "@type": "OfferCatalog",
    "name": "CNC Machining Capabilities",
    "itemListElement": [
      {
        "@type": "Offer",
        "itemOffered": {
          "@type": "Service",
          "name": "CNC Milling (3, 4, 5-Axis)"
        }
      },
      {
        "@type": "Offer",
        "itemOffered": {
          "@type": "Service",
          "name": "CNC Turning and Lathe Machining"
        }
      }
    ]
  },
  "description": "High-precision CNC machining services in China with tolerances down to ±0.005mm. Free DFM feedback and fast turnaround.",
  "offers": {
    "@type": "Offer",
    "priceCurrency": "USD",
    "price": "Contact for RFQ",
    "url": "https://www.yourdomain.com/services/cnc-machining/"
  }
}
```
